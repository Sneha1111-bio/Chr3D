# Chr3D Command Line Interface

Chr3D provides a powerful command-line interface for analyzing 3D chromatin conformation data from ChIA-PET, HiChIP, and Hi-C experiments.

## Installation

```bash
pip install chr3d
```

## Quick Start

```bash
# View all available commands
chr3d --help

# Run complete pipeline
chr3d run --mode <chiapet|hichip|hic> [options]

# Generate restriction sites
chr3d generate-sites --genome /path/to/genome.fa --enzyme MboI
```

## Available Commands

| Command | Description |
|---------|-------------|
| `run` | Run complete analysis pipeline |
| `generate-sites` | Generate restriction site file for HiChIP |

## Analysis Modes

Chr3D supports three analysis modes, each optimized for different experimental protocols:

### [ChIA-PET Mode](./chiapet)
For Chromatin Interaction Analysis by Paired-End Tag sequencing data. Includes linker filtering, mapping, and loop calling with statistical significance testing.

### [HiChIP Mode](./hichip)
For HiChIP (Hi-C + ChIP) data. Skips linker filtering (HiChIP doesn't use linkers), includes restriction fragment filtering, peak calling, and loop calling.

### [Hi-C Mode](./hic)
For standard Hi-C data. Generates contact matrices at multiple resolutions using pairtools and cooler.

## Global Options

```bash
chr3d [--version] [-v] [--log-file LOG_FILE] <command>
```

| Option | Description |
|--------|-------------|
| `--version` | Show version number |
| `-v, --verbose` | Enable verbose output |
| `--log-file` | Write logs to file |

## Pipeline Overview

```
┌─────────────────────────────────────────────────────────────────┐
│                     Chr3D Pipeline                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ChIA-PET:  FASTQ → Linker Filter → Map → Purify → Categorize   │
│                    → Peak Call → Loop Call → Significance        │
│                                                                  │
│  HiChIP:    FASTQ → Map → Purify (restriction) → Categorize     │
│                    → Peak Call → Loop Call → Significance        │
│                                                                  │
│  Hi-C:      FASTQ → Map → pairtools → cooler → Contact Matrix   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## Examples

### ChIA-PET Analysis
```bash
chr3d run --mode chiapet \
    --fastq-r1 sample_R1.fastq.gz \
    --fastq-r2 sample_R2.fastq.gz \
    --genome-index /path/to/hg38.fa \
    --linker-a ACGCGATATCTTATCTGACT \
    --linker-b AGTCAGATAAGATATCGCGT \
    --output-dir ./output \
    --threads 8
```

### HiChIP Analysis
```bash
chr3d run --mode hichip \
    --fastq-r1 sample_R1.fastq.gz \
    --fastq-r2 sample_R2.fastq.gz \
    --genome-index /path/to/hg38.fa \
    --restriction-sites /path/to/hg38_MboI_sites.bed \
    --output-dir ./output \
    --threads 8
```

### Hi-C Analysis
```bash
chr3d run --mode hic \
    --fastq-r1 sample_R1.fastq.gz \
    --fastq-r2 sample_R2.fastq.gz \
    --genome-index /path/to/hg38.fa \
    --chrom-sizes /path/to/hg38.chrom.sizes \
    --output-dir ./output \
    --threads 8
```
