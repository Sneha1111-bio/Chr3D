## Restriction Enzyme Digestion

All 3C-based methods rely on restriction enzyme digestion to fragment chromatin.

## Common Enzymes

| Enzyme | Recognition | Cut Frequency | Best For |
|--------|-------------|---------------|----------|
| MboI | ^GATC | ~700bp | Hi-C, HiChIP |
| HindIII | A^AGCTT | ~4kb | Hi-C |
| NcoI | C^CATGG | ~4kb | ChIA-PET |

## Generate Fragment Map

```python
from chr3d.utils import generate_restriction_map

frag_map = generate_restriction_map(
    genome="hg38",
    enzyme="MboI",
    output="hg38_MboI_fragments.bed"
)
```

## Assign Reads to Fragments

```python
hic_frag = hic.assign_fragments(fragments="hg38_MboI_fragments.bed")
```

## Quality Metrics

- **Valid pairs:** 40-60%
- **Self-circles:** &lt;5%
- **Dangling ends:** &lt;10%
- **Re-ligations:** &lt;20%

See guides for detailed fragment analysis.
