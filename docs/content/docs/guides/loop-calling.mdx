## Loop Calling Methods

Chromatin loops are focal enrichments in contact matrices representing stable physical interactions between genomic loci.

## Method Overview

| Method | Best For | Resolution | Computational Cost |
|--------|----------|------------|-------------------|
| **HiCCUPS** | Hi-C loops | 5-10kb | High |
| **HiChIPper** | HiChIP loops | 5-10kb | Medium |
| **ChIA-PET Tool** | ChIA-PET clusters | Variable | Low |
| **Fit-Hi-C** | Significant contacts | 10-40kb | Medium |

## Hi-C Loop Calling

### HiCCUPS Algorithm

```python
from chr3d import HiC

# Load normalized Hi-C data
hic = HiC.from_cooler("sample.cool")
hic_norm = hic.normalize(method="KR")

# Call loops with HiCCUPS
loops = hic_norm.call_loops(
    method="hiccups",
    resolution=10000,
    fdr=0.1,
    min_distance=20000,
    max_distance=2000000
)

print(f"Called {len(loops):,} loops")
```

**Parameters:**
- **resolution:** 5kb or 10kb recommended
- **fdr:** 0.1 (permissive) to 0.01 (stringent)
- **min_distance:** Typically 20kb (avoid self-ligation artifacts)
- **max_distance:** 2Mb (most loops are &lt;2Mb)

### Filtering Loops

```python
# Filter by enrichment
loops_high_conf = loops[loops['enrichment'] > 2.0]

# Filter by statistical significance
loops_sig = loops[loops['qvalue'] < 0.01]

# Filter by distance
loops_filtered = loops[
    (loops['distance'] > 20000) & 
    (loops['distance'] < 1000000)
]

print(f"High-confidence loops: {len(loops_high_conf):,}")
```

## HiChIP Loop Calling

### HiChIPper Algorithm

```python
from chr3d import HiChIP

# Load HiChIP data
hichip = HiChIP.from_pairs("sample.pairs.gz")

# Call loops with HiChIPper
loops = hichip.call_loops(
    method="hichipper",
    resolution=10000,
    fdr=0.01,
    min_distance=5000,
    max_distance=2000000,
    merge_distance=15000  # Merge nearby loops
)

print(f"Called {len(loops):,} loops")
```

**HiChIP-Specific Parameters:**
- **fdr:** 0.01 recommended (HiChIP has better signal-to-noise)
- **merge_distance:** 15kb (merge redundant calls)
- **min_distance:** 5kb (HiChIP can detect shorter loops)

### Peak-Anchored Calling

```python
from chr3d.utils import load_peaks

# Load ChIP-seq peaks
peaks = load_peaks("H3K27ac_peaks.bed")

# Require both anchors at peaks
loops_anchored = hichip.call_loops(
    method="hichipper",
    peaks=peaks,
    require_both_anchors=True,
    fdr=0.01
)

print(f"Peak-anchored loops: {len(loops_anchored):,}")
```

## ChIA-PET Loop Calling

### PET Cluster Method

```python
from chr3d import ChIAPET

# Load ChIA-PET data
chiapet = ChIAPET.from_bedpe("sample.bedpe")

# Call loops based on PET clusters
loops = chiapet.call_loops(
    min_pets=3,  # Minimum PETs per cluster
    peak_mode=True,  # Require anchor peaks
    fdr=0.01
)

print(f"Called {len(loops):,} PET clusters")
```

**Parameters:**
- **min_pets:** 2 (permissive), 3 (standard), 5+ (stringent)
- **peak_mode:** True (requires peaks at anchors)
- **extension:** Peak extension distance (default: 500bp)

### Peak Calling First

```python
# Call peaks at interaction anchors
peaks = chiapet.call_peaks(
    method="macs2",
    genome="hg38",
    qvalue=0.01
)

# Then call loops with peak requirement
loops = chiapet.call_loops(
    min_pets=3,
    peaks=peaks,
    peak_mode=True
)

print(f"Peaks: {len(peaks)}, Loops: {len(loops)}")
```

## Validation Methods

### Aggregate Peak Analysis (APA)

```python
# Compute APA score
apa = hic_norm.compute_apa(
    loops=loops,
    window=200000,
    resolution=10000
)

print(f"APA score: {apa['score']:.2f}")

# Plot APA
hic_norm.plot_apa(apa)
plt.savefig("apa.png", dpi=300)
```

**Interpretation:**
- **APA > 1.5:** Strong loop enrichment
- **APA 1.2-1.5:** Moderate enrichment
- **APA < 1.2:** Weak or no enrichment

### Motif Enrichment (CTCF)

```python
# For CTCF-mediated loops
loops_with_motifs = loops.add_motif_orientation(
    motif="CTCF",
    genome="hg38"
)

# Check convergent orientation
convergent = loops_with_motifs[
    loops_with_motifs['motif_orientation'] == 'convergent'
]

print(f"Convergent CTCF loops: {len(convergent)/len(loops_with_motifs)*100:.1f}%")
# Expected: 80-90% for CTCF loops
```

### Reproducibility Between Replicates

```python
# Load replicates
rep1_loops = hic1.call_loops(method="hiccups", fdr=0.1)
rep2_loops = hic2.call_loops(method="hiccups", fdr=0.1)

# Compute overlap
overlap = rep1_loops.overlap(rep2_loops, max_distance=20000)

print(f"Reproducible loops: {len(overlap):,}")
print(f"Reproducibility: {len(overlap)/len(rep1_loops)*100:.1f}%")
# Expected: >60% for good replicates
```

## Advanced Filtering

### Distance-Based Filtering

```python
# Remove very short-range (&lt;10kb)
loops_filtered = loops[loops['distance'] > 10000]

# Focus on specific distance range
enhancer_promoter = loops[
    (loops['distance'] > 10000) & 
    (loops['distance'] < 500000)
]
```

### Genomic Feature Filtering

```python
# Annotate with genomic features
loops_annotated = loops.annotate_genes(
    gtf="gencode.v38.annotation.gtf",
    promoter_window=2000
)

# Keep only promoter-involved loops
promoter_loops = loops_annotated[
    (loops_annotated['anchor1_feature'] == 'promoter') |
    (loops_annotated['anchor2_feature'] == 'promoter')
]

print(f"Promoter loops: {len(promoter_loops):,}")
```

### Blacklist Removal

```python
# Remove loops in problematic regions
loops_clean = loops.remove_blacklist(genome="hg38")

print(f"After blacklist removal: {len(loops_clean):,}")
```

## Multi-Resolution Loop Calling

```python
# Call loops at multiple resolutions
resolutions = [5000, 10000, 25000]
all_loops = []

for res in resolutions:
    hic_res = HiC.from_cooler(f"sample.mcool::resolutions/{res}")
    hic_res_norm = hic_res.normalize(method="KR")
    
    loops_res = hic_res_norm.call_loops(
        method="hiccups",
        resolution=res,
        fdr=0.1
    )
    
    all_loops.append(loops_res)
    print(f"{res/1000:.0f}kb: {len(loops_res):,} loops")

# Merge loops across resolutions
merged_loops = merge_multi_resolution_loops(all_loops)
```

## Differential Loop Analysis

```python
# Compare conditions
loops_treatment = hic_treatment.call_loops(method="hiccups", fdr=0.1)
loops_control = hic_control.call_loops(method="hiccups", fdr=0.1)

# Differential analysis
diff_loops = hic_treatment.differential_loops(
    loops_treatment=loops_treatment,
    loops_control=loops_control,
    method="deseq2"
)

# Significant changes
sig_diff = diff_loops[
    (diff_loops['padj'] < 0.05) & 
    (abs(diff_loops['log2fc']) > 1)
]

print(f"Gained loops: {len(sig_diff[sig_diff['log2fc'] > 0])}")
print(f"Lost loops: {len(sig_diff[sig_diff['log2fc'] < 0])}")
```

## Best Practices

### 1. Use Appropriate Resolution
- **5kb:** High-resolution, requires >1B reads
- **10kb:** Standard, requires >500M reads
- **25kb:** Lower resolution, requires >200M reads

### 2. Normalize Before Calling
```python
# Always normalize first
hic_norm = hic.normalize(method="KR")
loops = hic_norm.call_loops(method="hiccups")
```

### 3. Validate Results
```python
# Check APA score
apa = hic_norm.compute_apa(loops=loops)
assert apa['score'] > 1.2, "Weak loop enrichment"

# Check reproducibility
# (compare with replicate)
```

### 4. Filter Stringently
```python
# Multi-step filtering
loops_final = (loops
    .query('qvalue < 0.01')
    .query('enrichment > 2.0')
    .query('distance > 20000')
    .query('distance < 2000000')
)
```

## Troubleshooting

### Few Loops Called

**Causes:**
- Insufficient sequencing depth
- Poor data quality
- Too stringent parameters

**Solutions:**
```python
# Try more permissive FDR
loops = hic_norm.call_loops(method="hiccups", fdr=0.2)

# Lower distance threshold
loops = hic_norm.call_loops(min_distance=10000)

# Check data quality
qc = hic.qc_report()
```

### Too Many Loops

**Causes:**
- Too permissive parameters
- Poor normalization
- High background noise

**Solutions:**
```python
# More stringent FDR
loops = hic_norm.call_loops(method="hiccups", fdr=0.01)

# Filter by enrichment
loops = loops[loops['enrichment'] > 3.0]

# Check normalization
hic_norm.plot_matrix("chr1")  # Should look smooth
```

### Low APA Score

**Causes:**
- False positive loops
- Poor resolution
- Technical artifacts

**Solutions:**
```python
# Use higher resolution
hic_5kb = HiC.from_cooler("sample.mcool::resolutions/5000")

# More stringent filtering
loops = loops[loops['qvalue'] < 0.001]
```
