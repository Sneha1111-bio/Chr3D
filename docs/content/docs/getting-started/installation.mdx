## Installation

Install Chr3D using pip:

```bash
pip install chr3d
```

### Requirements

- Python 3.8+
- NumPy, Pandas, SciPy
- Cooler, Pysam

### Development Installation

```bash
git clone https://github.com/yourusername/chr3d.git
cd chr3d
pip install -e .
```

### Verify Installation

```python
import chr3d
print(chr3d.__version__)
```
