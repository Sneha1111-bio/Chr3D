## Choose Your Tutorial

### HiChIP Tutorials

HiChIP (Hi-C + ChIP) captures protein-centric chromatin interactions with low input requirements.

| Tutorial | Description |
|----------|-------------|
| [HiChIP API Tutorial](./hichip-api) | Complete Python API walkthrough with mermaid diagrams, ASCII logic explanations, class locations, and output structure |
| [HiChIP CLI Tutorial](./hichip-cli) | Command-line interface guide with step-by-step execution |

**Key Features:**
- No linker filtering (HiChIP doesn't use linkers)
- Restriction fragment-based purification
- Distance-corrected loop calling
- Peak calling from sPET

### Hi-C Tutorials

Hi-C captures genome-wide chromatin architecture without protein enrichment.

| Tutorial | Description |
|----------|-------------|
| [Hi-C API Tutorial](./hic-api) | Complete Python API walkthrough with pairtools and cooler integration |
| [Hi-C CLI Tutorial](./hic-cli) | Command-line interface guide for contact matrix generation |

**Key Features:**
- BWA-MEM alignment with Hi-C parameters
- pairtools processing pipeline
- Multi-resolution contact matrices (.mcool)
- QC statistics generation

## Quick Comparison

| Feature | HiChIP | Hi-C | ChIA-PET |
|---------|--------|------|----------|
| Linker filtering | No | No | Yes |
| Restriction sites | Yes | No | No |
| Peak calling | Yes | No | Yes |
| Loop calling | Built-in | External tools | Built-in |
| Output | Loops + Peaks | Contact matrix | Loops + Peaks |
| Input cells | ~1M | 1-10M | ~100M |

## Getting Started

### For HiChIP Analysis

```bash
# CLI approach
chr3d run --mode hichip \
    --fastq-r1 sample_R1.fastq.gz \
    --fastq-r2 sample_R2.fastq.gz \
    --genome-index /ref/hg38.fa \
    --restriction-sites /ref/hg38_MboI_sites.bed \
    --output-dir ./output
```

```python
# API approach
from chr3d.mapping_v2 import PETMapper
from chr3d.hichip_purifying import HiChIPPurifier
# ... see full tutorial
```

### For Hi-C Analysis

```bash
# CLI approach
chr3d run --mode hic \
    --fastq-r1 sample_R1.fastq.gz \
    --fastq-r2 sample_R2.fastq.gz \
    --genome-index /ref/hg38.fa \
    --chrom-sizes /ref/hg38.chrom.sizes \
    --output-dir ./output
```

