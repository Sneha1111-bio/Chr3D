## Complete Hi-C Analysis Workflow

This tutorial demonstrates comprehensive Hi-C data analysis using Chr3D.

## Load Data

```python
from chr3d import HiC

hic = HiC.from_cooler("K562_HiC.mcool::resolutions/10000")
```

## Quality Control

```python
qc = hic.qc_report()
print(f"Valid pairs: {qc['valid_pairs_pct']:.1f}%")
print(f"Cis/Trans: {qc['cis_trans_ratio']:.2f}")
```

## Normalize

```python
hic_norm = hic.normalize(method="ICE")
```

## Call Features

```python
# Compartments
compartments = hic_norm.call_compartments(resolution=100000)

# TADs
tads = hic_norm.call_tads(method="insulation", window=500000)

# Loops
loops = hic_norm.call_loops(method="hiccups", fdr=0.1)
```

## Validate Loops

```python
apa = hic_norm.compute_apa(loops=loops)
print(f"APA score: {apa['score']:.2f}")
```

## Export Results

```python
tads.to_bed("tads.bed")
loops.to_bedpe("loops.bedpe")
compartments.to_bedgraph("compartments.bedgraph")
hic_norm.to_hic("output.hic", resolutions=[10000, 25000])
```

See [API reference](../api/hic) for detailed method documentation.
