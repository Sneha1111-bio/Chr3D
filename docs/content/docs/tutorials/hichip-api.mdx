## Step-by-Step Tutorial

### Prerequisites

```python
import sys
import logging
from pathlib import Path

# Add Chr3D to path (if not installed)
sys.path.insert(0, '/path/to/Chr3D/src')

# Setup logging
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s'
)

# Import Chr3D modules
from chr3d.mapping_v2 import PETMapper
from chr3d.hichip_purifying import HiChIPPurifier
from chr3d.pet_categorization import PETCategorizer
from chr3d.peak_calling import PeakCaller
from chr3d.loop_calling import PreClusterer, AnchorClusterer, StatisticalSignificance
```

### Configuration

```python
# Input files
fastq_r1 = "/data/H3K27ac_HiChIP_R1.fastq.gz"
fastq_r2 = "/data/H3K27ac_HiChIP_R2.fastq.gz"
genome_index = "/ref/hg38.fa"
restriction_sites = "/ref/hg38_MboI_sites.bed"

# Output directory
output_dir = Path("/output/hichip_analysis")
output_dir.mkdir(parents=True, exist_ok=True)

# Create step directories
for step in ['step2_mapping', 'step3_purifying', 'step4_categorization', 
             'step5_peaks', 'step6_loops']:
    (output_dir / step).mkdir(exist_ok=True)

# Parameters
threads = 8
sample_id = "H3K27ac_HiChIP"
```

## Step 1: Skipped (No Linker Filtering)

**Important**: HiChIP does NOT use linker sequences. Unlike ChIA-PET, HiChIP uses biotinylated nucleotides during fill-in after restriction digestion. Therefore, we skip directly to mapping.

```
┌─────────────────────────────────────────────────────────────────┐
│  HiChIP vs ChIA-PET: Key Difference                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ChIA-PET:  Read ──[LINKER]── Read                              │
│             ↓                                                    │
│             Linker filtering required                            │
│                                                                  │
│  HiChIP:    Read ────────── Read                                │
│             ↓                                                    │
│             Direct mapping (no linker)                           │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## Step 2: Genomic Mapping

### What This Step Does

The `PETMapper` class aligns paired-end reads to the reference genome using BWA-MEM, then converts alignments to BEDPE format with deduplication.

### ASCII Diagram: Mapping Process

```
┌─────────────────────────────────────────────────────────────────┐
│                    STEP 2: MAPPING                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  R1.fastq.gz ─┐                                                 │
│               ├──► BWA-MEM ──► SAM ──► Sort ──► BAM             │
│  R2.fastq.gz ─┘      │                          │               │
│                      │                          │               │
│                      ▼                          ▼               │
│              Alignment Stats          Mark Duplicates           │
│                                              │                  │
│                                              ▼                  │
│                                    Parse to BEDPE               │
│                                              │                  │
│                                              ▼                  │
│                                    mapped.dedup.bedpe           │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

BEDPE Format (11 columns):
┌──────┬────────┬──────┬──────┬────────┬──────┬──────┬───────┬─────────┬─────────┬────────┐
│ chr1 │ start1 │ end1 │ chr2 │ start2 │ end2 │ name │ score │ strand1 │ strand2 │ mapq   │
├──────┼────────┼──────┼──────┼────────┼──────┼──────┼───────┼─────────┼─────────┼────────┤
│ chr1 │ 100000 │100150│ chr1 │ 200000 │200150│ read1│   60  │    +    │    -    │   30   │
└──────┴────────┴──────┴──────┴────────┴──────┴──────┴───────┴─────────┴─────────┴────────┘
```

### Code

```python
# =============================================================================
# STEP 2: GENOMIC MAPPING
# =============================================================================
print("=" * 70)
print("STEP 2: GENOMIC MAPPING")
print("=" * 70)

# Initialize mapper
mapper = PETMapper(
    genome_index=genome_index,
    mapping_quality_cutoff=30,    # Filter low-quality alignments
    n_threads=threads,
    use_bwa_mem=True,             # Recommended for HiChIP
    self_ligation_cutoff=8000     # Distance cutoff for self-ligation
)

# Run mapping
map_stats = mapper.map_linker_filtered_fastq(
    fastq_r1=fastq_r1,
    fastq_r2=fastq_r2,
    output_prefix='mapped',
    output_dir=str(output_dir / 'step2_mapping'),
    keep_sam=False,               # Delete SAM after conversion
    remove_duplicates=True        # Remove PCR duplicates
)

# Print statistics
print(f"Total read pairs: {map_stats.get('total_pets', 'N/A'):,}")
print(f"Mapped pairs: {map_stats.get('mapped_pets', 'N/A'):,}")
print(f"Unique PETs: {map_stats.get('unique_pets', 'N/A'):,}")
print(f"Duplicate rate: {map_stats.get('duplicate_rate', 0)*100:.1f}%")

bedpe_file = str(output_dir / 'step2_mapping/mapped.dedup.bedpe')
```

## Step 3: PET Purifying (Restriction Fragment Filtering)

### What This Step Does

The `HiChIPPurifier` removes same-fragment PETs using restriction site information. Same-fragment PETs are artifacts from self-ligation that don't represent real chromatin interactions.

### ASCII Diagram: Purification Logic

```
┌─────────────────────────────────────────────────────────────────┐
│                    STEP 3: PURIFYING                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Restriction Sites (MboI):                                       │
│  ────┬────────┬────────┬────────┬────────┬────────              │
│      │  Frag1 │  Frag2 │  Frag3 │  Frag4 │  Frag5               │
│      │        │        │        │        │                       │
│                                                                  │
│  For each PET:                                                   │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │ Read1 ────────────────────────────────── Read2          │    │
│  │   │                                         │           │    │
│  │   ▼                                         ▼           │    │
│  │ Assign to Fragment                  Assign to Fragment  │    │
│  │   │                                         │           │    │
│  │   └──────────── Compare ────────────────────┘           │    │
│  │                    │                                    │    │
│  │         ┌──────────┴──────────┐                         │    │
│  │         ▼                     ▼                         │    │
│  │   Same Fragment?        Different Fragment?             │    │
│  │         │                     │                         │    │
│  │         ▼                     ▼                         │    │
│  │   REMOVE (artifact)     KEEP (valid)                    │    │
│  │   → sameres.bedpe       → valid.bedpe                   │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

Example:
┌─────────────────────────────────────────────────────────────────┐
│ Fragment 1    Fragment 2    Fragment 3    Fragment 4            │
│ [────────]    [────────]    [────────]    [────────]            │
│                                                                  │
│ PET A: R1─────────────────────────────────R2  ✓ VALID           │
│        (Frag1)                        (Frag3)                    │
│                                                                  │
│ PET B: R1────R2                               ✗ SAME-FRAGMENT   │
│        (Frag2)(Frag2)                                            │
└─────────────────────────────────────────────────────────────────┘
```

### Code

```python
# =============================================================================
# STEP 3: PET PURIFYING (Restriction Fragment Filtering)
# =============================================================================
print("=" * 70)
print("STEP 3: PET PURIFYING")
print("=" * 70)

# Initialize purifier with restriction sites
purifier = HiChIPPurifier(restriction_file=restriction_sites)

# Define output files
purified_bedpe = str(output_dir / 'step3_purifying/purified.valid.bedpe')
sameres_bedpe = str(output_dir / 'step3_purifying/purified.sameres.bedpe')

# Run purification
purify_stats = purifier.remove_same_fragment_pets(
    bedpe_file,           # Input from Step 2
    purified_bedpe,       # Valid PETs output
    sameres_bedpe         # Removed same-fragment PETs
)

# Print statistics
print(f"Total PETs: {purify_stats.get('total_pets', 'N/A'):,}")
print(f"Valid PETs: {purify_stats.get('valid_pets', 'N/A'):,}")
print(f"Same-fragment removed: {purify_stats.get('same_fragment_pets', 'N/A'):,}")
print(f"Valid ratio: {purify_stats.get('valid_ratio', 0)*100:.1f}%")
```

### Expected Output


## Step 4: PET Categorization

### What This Step Does

The `PETCategorizer` classifies PETs into three categories based on genomic distance:
- **iPET (Inter-ligation)**: Long-range interactions (distance &gt; cutoff)
- **sPET (Self-ligation)**: Short-range/same-fragment (distance ≤ cutoff)
- **oPET (Other)**: Wrong orientation or unmapped

### ASCII Diagram: Categorization Logic

```
┌─────────────────────────────────────────────────────────────────┐
│                    STEP 4: CATEGORIZATION                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Distance Calculation:                                           │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │                                                          │    │
│  │  Read1 (chr1:100000)              Read2 (chr1:150000)   │    │
│  │     │                                  │                │    │
│  │     └──────── Distance = 50,000 bp ────┘                │    │
│  │                                                          │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Classification Rules (HiChIP: cutoff = 1000bp):                │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │                                                          │    │
│  │  Distance &gt; 1000bp AND same chromosome?                  │    │
│  │      YES → iPET (Inter-ligation PET)                    │    │
│  │                                                          │    │
│  │  Distance ≤ 1000bp AND same chromosome?                  │    │
│  │      YES → sPET (Self-ligation PET)                     │    │
│  │                                                          │    │
│  │  Different chromosomes?                                  │    │
│  │      YES → iPET (Trans interaction)                     │    │
│  │                                                          │    │
│  │  Invalid orientation or unmapped?                        │    │
│  │      YES → oPET (Other PET)                             │    │
│  │                                                          │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Output:                                                         │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐           │
│  │ iPET         │  │ sPET         │  │ oPET         │           │
│  │ (loops)      │  │ (peaks)      │  │ (discard)    │           │
│  │ → Step 6     │  │ → Step 5     │  │              │           │
│  └──────────────┘  └──────────────┘  └──────────────┘           │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Code

```python
# =============================================================================
# STEP 4: PET CATEGORIZATION
# =============================================================================
print("=" * 70)
print("STEP 4: PET CATEGORIZATION")
print("=" * 70)

# Initialize categorizer
# Note: HiChIP uses 1kb cutoff (shorter than ChIA-PET's 8kb)
categorizer = PETCategorizer(
    self_ligation_cutoff=1000,    # 1kb for HiChIP
    mode='hichip'
)

# Run categorization
cat_stats = categorizer.categorize_bedpe(
    purified_bedpe,               # Input from Step 3
    str(output_dir / 'step4_categorization/categorized')
)

# Print statistics
print(f"Total PETs: {cat_stats['total']:,}")
print(f"iPETs: {cat_stats['ipet']['count']:,} ({cat_stats['ipet']['percentage']:.1f}%)")
print(f"sPETs: {cat_stats['spet']['count']:,} ({cat_stats['spet']['percentage']:.1f}%)")
print(f"oPETs: {cat_stats['opet']['count']:,} ({cat_stats['opet']['percentage']:.1f}%)")
print(f"Cis ratio: {cat_stats['cis']['percentage']:.1f}%")
print(f"Trans ratio: {cat_stats['trans']['percentage']:.1f}%")

# Define output files for next steps
ipet_file = str(output_dir / 'step4_categorization/categorized.ipet')
spet_file = str(output_dir / 'step4_categorization/categorized.spet')
```

## Step 5: Peak Calling

### What This Step Does

The `PeakCaller` uses MACS3 to call enrichment peaks from self-ligation PETs (sPET). These peaks represent protein binding sites (e.g., H3K27ac marks for enhancers).

### Code

```python
# =============================================================================
# STEP 5: PEAK CALLING
# =============================================================================
print("=" * 70)
print("STEP 5: PEAK CALLING")
print("=" * 70)

# Initialize peak caller
peak_caller = PeakCaller(
    genome_size='hs',             # Human genome
    qvalue_cutoff=0.05,           # Q-value threshold
    keep_dup='all'                # Keep all reads (already deduplicated)
)

# Run peak calling
peak_stats = peak_caller.call_peaks(
    spet_file,                    # Input sPET from Step 4
    str(output_dir / 'step5_peaks/peaks')
)

# Print statistics
print(f"Peaks called: {peak_stats.get('num_peaks', 0):,}")
print(f"Peak file: {output_dir}/step5_peaks/peaks_peaks.narrowPeak")
```

## Step 6: Loop Calling

This is the most complex step, consisting of three sub-steps:

### Step 6.1: Pre-Clustering

### What This Step Does

The `PreClusterer` extends each PET tag by a specified length and creates initial cluster objects. This groups nearby interactions together.

### ASCII Diagram: Pre-Clustering Logic

```
┌─────────────────────────────────────────────────────────────────┐
│                    STEP 6.1: PRE-CLUSTERING                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Tag Extension (extension_length = 500bp):                       │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │                                                          │    │
│  │  Original tag position:                                  │    │
│  │       │                                                  │    │
│  │       ▼                                                  │    │
│  │  ─────●─────                                             │    │
│  │                                                          │    │
│  │  Forward strand (+): extend upstream                     │    │
│  │  ◄────●                                                  │    │
│  │  [pos-500, pos]                                          │    │
│  │                                                          │    │
│  │  Reverse strand (-): extend downstream                   │    │
│  │       ●────►                                             │    │
│  │  [pos, pos+500]                                          │    │
│  │                                                          │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Creating Cluster Objects:                                       │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │                                                          │    │
│  │  iPET Entry:                                             │    │
│  │  chr1:100000(+) ─────────────── chr1:200000(-)          │    │
│  │                                                          │    │
│  │  After Extension:                                        │    │
│  │  [99500-100000] ─────────────── [200000-200500]         │    │
│  │   (Anchor 1)                      (Anchor 2)             │    │
│  │                                                          │    │
│  │  Cluster Object:                                         │    │
│  │  {chr1, 99500, 100000, chr1, 200000, 200500, weight=1}  │    │
│  │                                                          │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Sort by chromosome and position, then merge overlapping         │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Code

```python
# =============================================================================
# STEP 6.1: PRE-CLUSTERING
# =============================================================================
print("=" * 70)
print("STEP 6.1: PRE-CLUSTERING")
print("=" * 70)

# Initialize pre-clusterer
pre_clusterer = PreClusterer(
    extension_length=500          # Extend each tag by 500bp
)

# Run pre-clustering
precluster_stats = pre_clusterer.pre_cluster(
    ipet_file,                    # Input iPET from Step 4
    str(output_dir / 'step6_loops/preclustered')
)

# Print statistics
print(f"Input iPETs: {precluster_stats.get('input_pets', 'N/A'):,}")
print(f"Pre-clusters created: {precluster_stats.get('cluster_count', 'N/A'):,}")
print(f"Output file: {precluster_stats.get('output_file', 'N/A')}")

precluster_file = precluster_stats['output_file']
```

### Step 6.2: Anchor Clustering

### What This Step Does

The `AnchorClusterer` merges overlapping pre-clusters into anchor clusters. Two clusters are merged if BOTH their head AND tail anchors overlap.

### ASCII Diagram: Anchor Clustering Logic

```
┌─────────────────────────────────────────────────────────────────┐
│                    STEP 6.2: ANCHOR CLUSTERING                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Merging Rule: Both anchors must overlap                         │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │                                                          │    │
│  │  Cluster A:  [====]──────────────────[====]             │    │
│  │              Anchor1                  Anchor2            │    │
│  │                                                          │    │
│  │  Cluster B:    [====]────────────────[====]             │    │
│  │                Anchor1                Anchor2            │    │
│  │                                                          │    │
│  │  Both overlap? ✓ Head overlaps, ✓ Tail overlaps         │    │
│  │  → MERGE into single cluster                             │    │
│  │                                                          │    │
│  │  Merged:     [======]────────────────[======]           │    │
│  │              (union)                  (union)            │    │
│  │              iPETs = A.iPETs + B.iPETs                   │    │
│  │                                                          │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Non-Merging Case:                                               │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │                                                          │    │
│  │  Cluster A:  [====]──────────────────[====]             │    │
│  │                                                          │    │
│  │  Cluster C:    [====]────────────────────────[====]     │    │
│  │                                                          │    │
│  │  Head overlaps? ✓                                        │    │
│  │  Tail overlaps? ✗                                        │    │
│  │  → DO NOT MERGE (different interactions)                 │    │
│  │                                                          │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
```

### Step 6.3: Statistical Significance

### What This Step Does

The `StatisticalSignificance` class calculates p-values for each loop using a distance-corrected Poisson model (for HiChIP) and applies Benjamini-Hochberg FDR correction.

### ASCII Diagram: Statistical Testing

```

│                    STEP 6.3: STATISTICAL SIGNIFICANCE            │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │

│  ┌─────────────────────────────────────────────────────────┐    │
│  │                                                          │    │

│  │                                                          │    │
```

## Complete Pipeline Script

Here's the complete script combining all steps:

```python
#!/usr/bin/env python3
"""
Complete HiChIP Analysis Pipeline using Chr3D Python API
"""

import sys
import logging
from pathlib import Path

sys.path.insert(0, '/path/to/Chr3D/src')

from chr3d.mapping_v2 import PETMapper
from chr3d.hichip_purifying import HiChIPPurifier
from chr3d.pet_categorization import PETCategorizer
from chr3d.peak_calling import PeakCaller
from chr3d.loop_calling import PreClusterer, AnchorClusterer, StatisticalSignificance

# Setup logging
logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

def run_hichip_pipeline(
    fastq_r1: str,
    fastq_r2: str,
    genome_index: str,
    restriction_sites: str,
    output_dir: str,
    threads: int = 8
):
    """Run complete HiChIP pipeline."""
    
    output_dir = Path(output_dir)
    output_dir.mkdir(parents=True, exist_ok=True)
    
    # Create step directories
    for step in ['step2_mapping', 'step3_purifying', 'step4_categorization',
                 'step5_peaks', 'step6_loops']:
        (output_dir / step).mkdir(exist_ok=True)
    
    # Step 2: Mapping
    mapper = PETMapper(genome_index=genome_index, n_threads=threads, use_bwa_mem=True)
    mapper.map_linker_filtered_fastq(fastq_r1, fastq_r2, 'mapped',
                                      str(output_dir / 'step2_mapping'))
    
    # Step 3: Purifying
    purifier = HiChIPPurifier(restriction_file=restriction_sites)
    purifier.remove_same_fragment_pets(
        str(output_dir / 'step2_mapping/mapped.dedup.bedpe'),
        str(output_dir / 'step3_purifying/purified.valid.bedpe'),
        str(output_dir / 'step3_purifying/purified.sameres.bedpe')
    )
    
    # Step 4: Categorization
    categorizer = PETCategorizer(self_ligation_cutoff=1000, mode='hichip')
    categorizer.categorize_bedpe(
        str(output_dir / 'step3_purifying/purified.valid.bedpe'),
        str(output_dir / 'step4_categorization/categorized')
    )
    
    # Step 5: Peak Calling
    peak_caller = PeakCaller(genome_size='hs')
    peak_caller.call_peaks(
        str(output_dir / 'step4_categorization/categorized.spet'),
        str(output_dir / 'step5_peaks/peaks')
    )
    
    # Step 6: Loop Calling
    pre_clusterer = PreClusterer(extension_length=500)
    precluster_stats = pre_clusterer.pre_cluster(
        str(output_dir / 'step4_categorization/categorized.ipet'),
        str(output_dir / 'step6_loops/preclustered')
    )
    
    anchor_clusterer = AnchorClusterer()
    anchor_clusterer.cluster_anchors(
        precluster_stats['output_file'],
        str(output_dir / 'step6_loops/clusters.txt')
    )
    
    stat_sig = StatisticalSignificance(use_distance_correction=True)
    stat_sig.calculate_significance(
        str(output_dir / 'step6_loops/clusters.txt'),
        str(output_dir / 'step4_categorization/categorized.ipet'),
        str(output_dir / 'step6_loops/loops')
    )
    
    print(f"Pipeline complete! Results in: {output_dir}")

if __name__ == '__main__':
    run_hichip_pipeline(
        fastq_r1='/data/sample_R1.fastq.gz',
        fastq_r2='/data/sample_R2.fastq.gz',
        genome_index='/ref/hg38.fa',
        restriction_sites='/ref/hg38_MboI_sites.bed',
        output_dir='/output/hichip_results',
        threads=8
    )
```
