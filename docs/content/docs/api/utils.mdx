## Utility Functions

Chr3D provides utility functions for common tasks in chromatin conformation analysis.

## Key Functions

### File I/O
- `load_peaks()` - Load peaks from BED format
- `load_genes()` - Load gene annotations from GTF
- `load_bedpe()` - Load interactions from BEDPE

### Genome Utilities
- `generate_restriction_map()` - Generate restriction fragment map
- `get_chromosome_sizes()` - Get chromosome sizes
- `load_blacklist()` - Load ENCODE blacklist regions

### Annotation
- `annotate_genes()` - Annotate regions with genes
- `annotate_features()` - Annotate with genomic features
- `find_overlaps()` - Find overlaps between regions

See guides for detailed usage examples.
