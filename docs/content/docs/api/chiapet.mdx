# ChIA-PET Python API

Complete Python API for analyzing ChIA-PET (Chromatin Interaction Analysis by Paired-End Tag sequencing) data.

## Overview

ChIA-PET analysis requires these modules:
- `LinkerFilterV3` - Linker detection and filtering
- `PETMapper` - Read mapping
- `ChIAPETPurifier` - Duplicate removal and merging
- `PETCategorizer` - PET classification
- `PeakCaller` - Peak calling from sPET
- `PreClusterer`, `AnchorClusterer`, `StatisticalSignificance` - Loop calling

## Complete Pipeline Example

```python
import sys
sys.path.insert(0, '/path/to/Chr3D/src')

from pathlib import Path
from chr3d.linker_filtering_v3 import LinkerFilterV3
from chr3d.mapping_v2 import PETMapper
from chr3d.chiapet_purifying import ChIAPETPurifier
from chr3d.pet_categorization import PETCategorizer
from chr3d.peak_calling import PeakCaller
from chr3d.loop_calling import PreClusterer, AnchorClusterer, StatisticalSignificance

# Configuration
fastq_r1 = "/data/sample_R1.fastq.gz"
fastq_r2 = "/data/sample_R2.fastq.gz"
genome_index = "/ref/hg38.fa"
output_dir = Path("/output/chiapet_analysis")
output_dir.mkdir(parents=True, exist_ok=True)

# =============================================================================
# STEP 1: LINKER FILTERING
# =============================================================================
linker_filter = LinkerFilterV3(
    linker_sequences=['ACGCGATATCTTATCTGACT', 'AGTCAGATAAGATATCGCGT'],
    min_alignment_score=14,
    min_tag_length=18,
    max_tag_length=1000,
    n_threads=8
)

linker_stats = linker_filter.filter_fastq_parallel(
    fastq_r1=fastq_r1,
    fastq_r2=fastq_r2,
    output_prefix='filtered',
    output_dir=str(output_dir / 'step1_linker')
)

print(f"Valid PETs after linker filtering: {linker_stats['valid_pets']:,}")

# =============================================================================
# STEP 2: GENOMIC MAPPING
# =============================================================================
mapper = PETMapper(
    genome_index=genome_index,
    mapping_quality_cutoff=30,
    n_threads=8,
    use_bwa_mem=True,
    self_ligation_cutoff=8000
)

# Use filtered FASTQ files
filtered_r1 = str(output_dir / 'step1_linker/filtered.1_1.R1.fastq')
filtered_r2 = str(output_dir / 'step1_linker/filtered.1_1.R2.fastq')

map_stats = mapper.map_linker_filtered_fastq(
    fastq_r1=filtered_r1,
    fastq_r2=filtered_r2,
    output_prefix='mapped',
    output_dir=str(output_dir / 'step2_mapping'),
    keep_sam=False,
    remove_duplicates=True
)

print(f"Unique PETs: {map_stats.get('unique_pets', 'N/A'):,}")

# =============================================================================
# STEP 3: PET PURIFYING
# =============================================================================
purifier = ChIAPETPurifier(merge_distance=2)

bedpe_file = str(output_dir / 'step2_mapping/mapped.dedup.bedpe')
purify_stats = purifier.purify(
    bedpe_file,
    str(output_dir / 'step3_purifying/purified')
)

purified_bedpe = str(output_dir / 'step3_purifying/purified.merged.bedpe')

# =============================================================================
# STEP 4: PET CATEGORIZATION
# =============================================================================
categorizer = PETCategorizer(
    self_ligation_cutoff=8000,  # 8kb for ChIA-PET
    mode='chiapet'
)

cat_stats = categorizer.categorize_bedpe(
    purified_bedpe,
    str(output_dir / 'step4_categorization/categorized')
)

print(f"iPETs: {cat_stats['ipet']['count']:,}")
print(f"sPETs: {cat_stats['spet']['count']:,}")

ipet_file = str(output_dir / 'step4_categorization/categorized.ipet')
spet_file = str(output_dir / 'step4_categorization/categorized.spet')

# =============================================================================
# STEP 5: PEAK CALLING
# =============================================================================
peak_caller = PeakCaller(
    genome_size='hs',
    qvalue_cutoff=0.05,
    keep_dup='all'
)

peak_stats = peak_caller.call_peaks(
    spet_file,
    str(output_dir / 'step5_peaks/peaks')
)

print(f"Peaks called: {peak_stats.get('num_peaks', 0):,}")

# =============================================================================
# STEP 6: LOOP CALLING
# =============================================================================

# Step 6.1: Pre-clustering
pre_clusterer = PreClusterer(extension_length=500)
precluster_stats = pre_clusterer.pre_cluster(
    ipet_file,
    str(output_dir / 'step6_loops/preclustered')
)

# Step 6.2: Anchor clustering
anchor_clusterer = AnchorClusterer()
cluster_stats = anchor_clusterer.cluster_anchors(
    precluster_stats['output_file'],
    str(output_dir / 'step6_loops/clusters.txt')
)

# Step 6.3: Statistical significance
stat_sig = StatisticalSignificance(
    ipet_count_threshold=2,
    pvalue_cutoff=0.05,
    extension_length=500,
    use_distance_correction=False  # Standard Poisson for ChIA-PET
)

sig_stats = stat_sig.calculate_significance(
    str(output_dir / 'step6_loops/clusters.txt'),
    ipet_file,
    str(output_dir / 'step6_loops/loops')
)

print(f"Significant loops: {sig_stats.get('significant_loops', 0):,}")
```

## Module Reference

### LinkerFilterV3

High-performance linker filtering using Smith-Waterman alignment.

```python
from chr3d.linker_filtering_v3 import LinkerFilterV3

filter = LinkerFilterV3(
    linker_sequences: List[str],      # Linker sequences to detect
    min_alignment_score: int = 14,    # Minimum alignment score
    min_tag_length: int = 18,         # Minimum tag length after trimming
    max_tag_length: int = 1000,       # Maximum tag length
    n_threads: int = 1                # Number of threads
)

# Main method
stats = filter.filter_fastq_parallel(
    fastq_r1: str,           # Input R1 FASTQ
    fastq_r2: str,           # Input R2 FASTQ
    output_prefix: str,      # Output file prefix
    output_dir: str          # Output directory
) -> Dict
```

**Returns:**
```python
{
    'total_reads': int,
    'valid_pets': int,
    'linker_1_1': int,  # Both reads have linker A
    'linker_2_2': int,  # Both reads have linker B
    'linker_1_2': int,  # Chimeric
    'no_linker': int
}
```

### PETMapper

BWA-based read mapping with BEDPE generation.

```python
from chr3d.mapping_v2 import PETMapper

mapper = PETMapper(
    genome_index: str,                    # BWA-indexed genome
    mapping_quality_cutoff: int = 30,     # Min MAPQ
    n_threads: int = 1,                   # Threads
    use_bwa_mem: bool = False,            # Use BWA-MEM vs BWA-ALN
    self_ligation_cutoff: int = 8000      # Self-ligation distance
)

# Main method
stats = mapper.map_linker_filtered_fastq(
    fastq_r1: str,
    fastq_r2: str,
    output_prefix: str,
    output_dir: str = None,
    keep_sam: bool = False,
    remove_duplicates: bool = True
) -> Dict
```

### ChIAPETPurifier

Removes duplicates and merges overlapping PETs.

```python
from chr3d.chiapet_purifying import ChIAPETPurifier

purifier = ChIAPETPurifier(
    merge_distance: int = 2    # Max distance for merging
)

stats = purifier.purify(
    input_bedpe: str,
    output_prefix: str
) -> Dict
```

### PETCategorizer

Classifies PETs into categories.

```python
from chr3d.pet_categorization import PETCategorizer

categorizer = PETCategorizer(
    self_ligation_cutoff: int = 8000,  # Distance cutoff
    mode: str = 'chiapet'              # 'chiapet' or 'hichip'
)

stats = categorizer.categorize_bedpe(
    input_bedpe: str,
    output_prefix: str
) -> Dict
```

**Returns:**
```python
{
    'total': int,
    'ipet': {'count': int, 'percentage': float},
    'spet': {'count': int, 'percentage': float},
    'opet': {'count': int, 'percentage': float},
    'cis': {'count': int, 'percentage': float},
    'trans': {'count': int, 'percentage': float}
}
```

### PeakCaller

MACS3-based peak calling.

```python
from chr3d.peak_calling import PeakCaller

caller = PeakCaller(
    genome_size: str = 'hs',       # 'hs', 'mm', or number
    qvalue_cutoff: float = 0.05,   # Q-value threshold
    keep_dup: str = 'all'          # Duplicate handling
)

stats = caller.call_peaks(
    input_file: str,
    output_prefix: str
) -> Dict
```

### Loop Calling Classes

#### PreClusterer

```python
from chr3d.loop_calling import PreClusterer

clusterer = PreClusterer(
    extension_length: int = 500,        # Tag extension
    chrom_sizes_file: str = None        # Optional chrom sizes
)

stats = clusterer.pre_cluster(
    ipet_file: str,
    output_prefix: str
) -> Dict
```

#### AnchorClusterer

```python
from chr3d.loop_calling import AnchorClusterer

clusterer = AnchorClusterer()

stats = clusterer.cluster_anchors(
    precluster_file: str,
    output_file: str
) -> Dict
```

#### StatisticalSignificance

```python
from chr3d.loop_calling import StatisticalSignificance

sig = StatisticalSignificance(
    ipet_count_threshold: int = 2,      # Min iPET count
    pvalue_cutoff: float = 0.05,        # P-value threshold
    extension_length: int = 500,        # Extension for counting
    use_distance_correction: bool = False  # Distance correction (HiChIP)
)

stats = sig.calculate_significance(
    cluster_file: str,
    ipet_file: str,
    output_prefix: str
) -> Dict
```

## Output Files

| File | Description |
|------|-------------|
| `filtered.*.R1.fastq` | Linker-filtered R1 reads |
| `filtered.*.R2.fastq` | Linker-filtered R2 reads |
| `mapped.dedup.bedpe` | Deduplicated mapped PETs |
| `purified.merged.bedpe` | Merged PETs |
| `categorized.ipet` | Inter-ligation PETs |
| `categorized.spet` | Self-ligation PETs |
| `peaks_peaks.narrowPeak` | Called peaks |
| `loops.cluster.FDRfiltered.txt` | Significant loops |

## Common Linker Sequences

| Protocol | Linker A | Linker B |
|----------|----------|----------|
| ChIA-PET Tool | `ACGCGATATCTTATCTGACT` | `AGTCAGATAAGATATCGCGT` |
| ENCODE | `GTTGGATAAG` | `GTTGGAATGT` |
