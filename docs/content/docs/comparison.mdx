## Quick Comparison

| Feature | Hi-C | ChIA-PET | HiChIP |
|---------|------|----------|---------|
| **Input cells** | 1-10M | 10-50M | 100K-1M |
| **Protein specificity** | No | Yes | Yes |
| **Cost** | $$ | $$$$ | $$ |
| **Best for** | Genome architecture | Specific proteins | Low-input loops |

## When to Use Each

### Hi-C
Use for genome-wide chromatin architecture, TAD identification, and compartment analysis.

### ChIA-PET
Use for high-confidence protein-specific loops with abundant sample material.

### HiChIP
Use for protein-specific interactions with limited input material or large cohort studies.
